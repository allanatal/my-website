BSA and Capecitabine Dose Calculator

Calculates BSA from height and weight (or uses a given BSA) and suggests Capecitabine doses and tablet counts for two scenarios.

For adults only. Capecitabine tablets assumed to be 500 mg each. This tool is for educational support and does not replace clinical judgment.
Patient data and BSA
If BSA is provided, height and weight are not required.
If BSA is not provided, BSA will be calculated using the Mosteller formula.
This is the BSA that will be used for dose calculations.

Scenario 1 – Capecitabine with chemoRT

Regimen: Capecitabine 825 mg/m² PO BID, Monday–Friday, on each day that RT is given throughout the duration of RT.

Preselected value is 30. Change if needed.

Scenario 2 – Capecitabine (systemic)

Regimen options: dose per day based on BSA (total daily dose).

Total daily dose; the calculator will split tablets into morning and night.
Exact tablets/day (with decimals) will be displayed; prescription uses the rounded value.
